win10一些装逼技巧

发表于 2017-09-01

这篇文章来总结一下怎么给你一个使用起来很装逼的系统！

win10 任务栏透明方法

  1. win+r键 输入regedit 打开注册表编辑器，找到位置:
    \HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced

  2. 右键–>新建–>DWORD(32位)值 ，然后重命名为 UseOLEDTaskbarTransparency ，双击，将数值数据改为1

  3. 重启电脑，或者注销再打开，或者打开任务管理器，关闭 资源管理器 进程，然后点文件，运行 输入 explorer 即可。

记得要在设置 –> 个性化 –> 颜色中 把透明效果打开，不然无用。

win10 任务栏图标居中方法

  1. 在桌面新建一个文件夹
  2. 任务栏右键 -> 取消锁定所有任务栏 –> 工具栏 –> 新建工具栏
  3. 然后选择桌面刚才新建的文件夹，在任务栏右边就会出现一栏
  4. 将其移动到最左边的图标左边，并拖动长度，使你的图标处于居中位置
  5. 右键这个工具栏，取消显示标题
  6. 锁定任务栏，大功告成。

此方法有一个弊端，就是当你新打开一个软件时，如果任务栏没有固定该图标，那么就回新加一个图标，所以就不会居中了~~
